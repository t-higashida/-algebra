\subsection{素体・標数}

可換体$K$のすべての部分体の共通部分はまた$K$の部分体になる．これを$K$の\textbf{素体}という．

\定理{13}
素体は$F_p = \Z / p \Z$（$p$は素数）または有理数体$\Q$と同型である．

\証明

可換体$K$の素体を$P$とすれば，$P \in 0 , 1 $、従って任意の$n \in \Z$に対して、
$n 1 \in P$．今$f : \Z \to P$を$ f(n) = n1$によって定義すれば，\underline{$ f$はは$\Z$から$P$の中への環としての準同型}となる．
その核は$ \Z$の ideal であるから，ある整数$p$を用いて $ \Ker f = p\Z$と表わされる．

$ \text{Im } f $は$P$の部分環であるから整域であり，第1同型定理により$\Z / p \Z \cong \textbf{Im} f$であるから，
$ \Z / p \Z$も整域となる．従って \S 2.3 例2により，$p$は合成数ではあり得ない．
すなわち，$p$は素数または$0$または$1$となる．
$ p $が素数ならば，$F_p \cong \Z / p \Z \cong \Z / p \cong \text{Im }$により
\underline{$ \text{Im} f $は$P$に含まれる体となる．}
ゆえに$ P$の定義により，$P = \text{Im } f \cong F_o$．
$P$の定義により，$ P = \text{Im} f \cong F_p$．

$p = 0$ならば$ \Z \cong \text{Im } f \subset P$．ゆえに$ |Q \cong ( \text{Im} f \text{の商体} ) \subset P$，
従って$P$の定義により，
$ P = ( \text{Im} f \text{の商体}) \cong \Q$．

$p=1$ならば，$\Ker f \ni 11$．これは$f$の定義に反する，ゆえにこの場合はおこらない．
\■

可換体$K$の素体が$F_p$と同型であるとき，$K$の\textbf{標数}は$P$であるといい，
素体が$\Q$と同型であるとき，$K$の標数は$0$であるという．

\例{1}

$F_p$と同型な体の標数は$p$である．この体は$p$個の元からなり，$p$元体とよぶ．


\例{2}
$\Q,\R,\C$の標数は$0$である．

\問{1} 標数$p$の可換体$K$において，
\begin{align*}
	p \cdot a = \underbrace{a + a + \cdots + a}_{\text{$p$個}}= 0 \quad ( a \in K).	\\
	( a \pm b )^{p^e} = a^{p^e} \pm b^{b^e} \quad ( a,b \in K , e \in \N)	
\end{align*}
が成り立つ．

\証明
$ p \cdot a = a(1 + 1 + \cdots + 1 ) = a \cdot 0 = 0$．また二項係数$\displaystyle \binom{p^e}{r}$は$1 \leqq r < p^e -1$に対しては，
$p$の倍数だかあ，$K$において$ \displaystyle \binom{p^e}{r}x = 0$．ゆえに二項定理により，第2式が得られる．
\■

\問{2}

標数$p$の可換体$K$において，方程式$x^{p^e}=c \quad ( c \in K,e \in \N)$の根は
たかだか一つであることを証明せよ．

この根があるとき，それを$c$の$p^e$乗根といい，$ c^{1/p^e}$で表わす．$ a , b $の$p^e$乗根があるとき，
$a \pm b$の$p^e$乗根もあって
\[
	a^{1/p^e} \pm b^{1/p^e} = ( a \pm b)^{1/p^e}
\]
となることを証明せよ．

\証明

$x^{p^e} = y^{p^e} \Rightarrow x^{p^e} - y^{p^e} = 0 \Rightarrow (x-y)^{p^e} = 0 \Rightarrow x-y$．
後半は
$ a^{1/p^e} \pm b^{1/p^e} = a \pm b$から明らか．
 

\問{3} 
$p$元体$K$では，$ f: K \to K^{p^e} (f(x)=x^{p^e})$は同型写像である．したがってすべての元が$ p^e$乗根をもつ．

\証明
問2より，$f$は単射である．したがって$K^{p^e}$の元の数は$K$の元の数に等しく$f$は全射となる．

\■


\問{4}
有限可換体の標数は$0$でない．

\証明

可換体$K$の標数が$0$ならば，$1, 1 + 1 , 1+1+ 1 , \cdots $はすべて互いに異なり，
$K$が無限に多くの元をもつ．

\■


〔注意〕有限体の構造についてさらにくわしくは,第VI章 \S 2で扱う．
