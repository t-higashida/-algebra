\subsection{準同型写像と同型定理}

群$G$から群$G'$への写像$f$が，$G$の任意の2元$x,y$に対して
$f(xy)=f(x) \cdot f(y)$をみたすとき，
$f$は\textbf{演算を保存する}あるいは\textbf{準同型（写像）}であるという．
$G'$の単位元を$e'$とするとき，$ \{ \: x \in G \: ;  \: f(x)=e' \: \}$を
準同型$f$の\textbf{核}（kernel）といい，$ \Ker f$で表わす．


\問{1}
準同型写像$f : G \to G'$について次のおのおのが成り立つ．
ただし$e,e'$はそれぞれ$G,G'$の単位元とする．
\begin{list}{}{}
	\item (1) $ f(e) = e'$．	
	\item (2) $f(x^{-1}) = (f(x))^{-1}$．
	\item (3) $\text{Im} f = f(G)$は$G'$の部分群である．
	\item (4) $\Ker f $は$G$の正規部分群である．――
\end{list}
\証明
(1) $ f(e) f(e) = f(ee) = f(e)$．両辺に$f(e)^{-1}$を掛けて，$f(e)=e'$．

(2) $ f(x) f(x^{-1}) = f(xx^{-1}) = f(e) = e'$．ゆえに$f(x^{-1} ) = f(x)^{-1}$．

(3) $ f(x) f(y)^{-1} = f(x) f(y^{-1}) = f(xy^{-1})$．ゆえに$f(G)f(G)^{-1} \subset f(G)$．従って
$ f(G)$は部分群である．

(4) $f(x) = e' , \: f(y)=e'$ならば$ f(xy^{-1}) = f(x)f(y)^{-1} = e'$．従って
$ \Ker f$は$G$の部分群である．また$G$の任意の元$a$に対して，$f(axa^{-1}) f(a) f(x) f(a^{-1}) = f(a) e' f(a)^{-1} = e'$．
ゆえに $\Ker f$は$G$の正規部分群である．

\■


群$G$から群$G'$への準同型全射があるとき，
$G'$は$G$に\textbf{準同型}であるという．また，準同型単射を（中ヘの）\textbf{同型（写像）}という．
$ f : G \to G'$が同型写像ならば，$G$の各元$x$と$f(x) \in G'$とを同一視して，
$G$が$G'$の中へ\textbf{うめこまれる（imbed される）}という．
$G$から$G'$への上への同型写像があるとき，$G$と$G'$とは\textbf{同型}であるといい，
$ G \cong G'$と書く．とくに$G$から$G$自身の中への準同型写像，
同型全射をそれぞれ$G$の\textbf{自己準同型}，\textbf{自己同型}という．

\問{2}
準同型写像$f:G \to G'$が中への同型写像であるためには，
$ \Ker f = \{ e \}$の成り立つことが必要かつ十分である．

\証明
$ \Ker f = \{ e \}$ならば，$f(x) = f(y) \Rightarrow f(x)f(y^{-1}) = f(xy') = e' \Rightarrow xy-{-1} = e \Rightarrow x = y$
となり$f$は単射となる．逆は明らかである．

\例{1}
群$G$の恒等写像は，$G$のの自己同型である．また，$G$のすべての元に$G'$の単位元$e'$を対応させる写像$f$は，
準同型写像で，$ \text{Im} f = \{ e' \} , \Ker f = G$である．


\例{2}
$N$を群$G$の正規部分群とするとき，標準的全射
$ G \to G / N ( x \to x \mod N)$は
準同型全射である．これを\textbf{標準的準同型}とよぶ．


\例{3}
$a$を一つの実数とする．任意の実数$x$に$ax$対応させる写像$f$は，実数の
加法群$\R$から$\R$への準同型写像である．$a = 0$ならば$f$は
$ \text{Im } f = \{ 0 \}, \Ker f = \R$の自己準同型である．
$a \ne 0$ならば$f$は全単射でしたがって$\R$の自己同型である．

\例{4} $0$でない複素数全体の作る乗法群を$ \C^*$とする．
$ \C^*$の各元にその絶対値を対応させる写像$f$は，
$ \C^{*}$から$ \C^*$への準同型写像である．
 $ \text{Im } f$は正の実数全体の作る乗法群$ \R^+$で，
$\Ker f$は絶対値$1$の複素数全体の集合である．
この準同型写像により，$ \R^+$は$\C^*$に準同型である．

\例{5}
$ \C^*$の各元を2乗する写像は$ \C^*$から$ \C^*$の上への準同型写像である．
各は $ \{ \: 1, -1 \: \}$．

\例題{}
$ f : G \to G / N$を標準的準同型とすれば，$N$を含む$G$の部分群全体と
$G /N$の部分群全体とは，$ H \to f(H)$によって 1対1に対応する．

\証明
$ G / N$の部分群$H'$に対して，$ f^{-1} (H')$は$N$を含む$G$の部分群となる．
実際，$f^{-1} (H') \ni x , y \Rightarrow f(x) ,f(y) \in H' \Rightarrow f(xy^{-1}) = f(x) f(y)^{-1} \in H'
 \Rightarrow xy^{-1} \in f^{-1} (H')$．また，$f(N)= \{ e' \} \subset H'$ゆえ$ N \subset f^{-1} (H')$．

そこで，$ff^{-1} (H') = H'$，$f^{-1} f(H) = H$がつねに成り立つことを示せばよい（p.4 定理3）

まず，$ f f^{-1} (H') , f^{-1} f(H) \supset H$はp.3定理2による．逆に，$f^{-1}f(H) \ni x \Rightarrow f(x) \in f(H) 
\Rightarrow \exists h \in H$，
$ f(x) = f(h) \Rightarrow f(xh^{-1}) = e \Rightarrow xh^{-1} \in N \subset H \Rightarrow x \in hH = H$．
ゆえに$f^{-1} \cdot f(H) \subset H$．

\問{3}
整数の加法群$\Z$の自己準同型$f$は，かならず$ f(x) = ax \: (a \in \Z)$の形に表わされる．
かつ $ a \ne 0$ならば，これは中への同型写像となる．

\証明
$ f(1) =a$とおけば，正の整数$x$に対しては，$f(x) = f( \underbrace{1 + \cdots + 1}_{x\text{個}})
= \underbrace{ f(1) + \cdots + f(1)}_{ x \text{個}} = ax$．
負の整数$ x = -y$に対して，$f(x) = - f(y) = -a y = ax$．

また $f(0) = 0  = a 0$．ゆえに任意の$x \in \Z$に対して$f(x) = ax$となる．$a \ne 0$ならば，
$ \Ker f = \{ \: x \in \Z \: ; \: ax = 0 \: \} = \{ \: 0 \: \}$．

\問{4}
準同型写像の結合は準同型写像である．同型写像の結合は同型写像である．
同型全射の逆写像も同型全射である．

\証明
$f :G \to G' , g: G' \to G''$がともに準同型写像ならば $ ( g \circ f) (xy) = g( f(x) f(y)) = (gf(x)) (gf(y))$．
ゆえに$ g \circ f$も準同型写像である．他の部分は，一般に単射の結合が単射であること，全単射の逆写像が全単射であることから
明らかである．

$f,g$が$G$の自己同型ならば$g \circ f, f^{-1}$もまた$G$の自己同型となることが，問4および

\定理{4}
（第1同型定理）準同型 $f : G \to G'$の核を$N$とすれば，
$ G / N \cong f(G) $．

かつこの同型は，$x \mod N \leftrightarrow f(x)$によって与えられる．

\定理{5}
（第2同型定理）
$H$を$G$の部分群，$N$を$G$の正規部分群とすれば，
\[
	H / ( N \cap H) \cong N H / N.
\]
かつこの同型は，$ x \mod ( N \cap H) \to x \mod N ( x \in H )$によって与えられる．

\証明
\S 1.3 の 例題により両辺は意味をもつ．$x \mod ( N \cap H ) \rightarrow x \mod N \: ( x \in H) $によって，
(1) 写像が $x$の選び方によらず定義され，(2) 準同型となり，(3) 全射となり，(4) 核が単位元となることは容易に検証される．

\問{8} 定理5 証明の (1)〜(4)を検証せよ．

\証明
$G$から$G/N$への準同型写像$p$によって，$G$の部分群$H$は$G/N$の部分群$HN/N$に写される．
そこで$p$を$H$から$HN/N$の上への準同型写像と見なすと，$ \Ker p = H \cap N$より
$ H / ( \Ker p) \cong H / (N [cap N) \cong \text{Im} p \cong HN/N$．
