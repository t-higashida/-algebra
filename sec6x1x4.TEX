\subsection{Jordan-H\"{o}1derの定理}

本稿では，作用域$ \Omega$をもつ群$G$を考える．$ \Omega = \varnothing$の場合を考えれば，本稿の所論は
作用域を考えない群にも適用される．

$G$を認容部分群の列
\[
	G = G_0 \supset G_1 \supset \cdots \supset G_n = \{ e \}
\]
において，$G_{i+1}$が$G_i$の認容正規部分群であるとき，
$ \{ G_i \}_{0 < i \leqq n}$を
$ ( G , \Omega )$の\textbf{正規列}という．$G_i \supsetneq G_{i+1} \: ( 0 \leqq i \leqq n-1)$のとき，正規列は\textbf{簡約されている}という．
もし，$G_i = G_{i+1}$ならば，どちらかを省くことによって正規列を簡約することができる．二つの正規列
$\{ G_i \: \}_{ 0 \leq i \leq n}, \{ H_j \}_{0 \leq j \leq m}$においてすべての$G_i$が
$H_0 , H_1 , \cdots , H_m$の中に現れるとき，$ \{ H_j \}$を$ \{ G_i \}$の\textbf{細分}であるという．
このときさらに少なくとも一つの$H_i$が$G_0 , G_1 , \cdots , G_n$の中に現れないならば，
$ \{ H_i \}$を$ \{ iG_i \}$の\textbf{真の細分}であるという．簡約された正規列$ \{ G_i \}_{0 \leq i \leq n}$が
真の細分を持たないとき，これを$G$の\textbf{組成列}という．

二つの正規列$ \{ G_i \}_{ 0 \leq i \leq n}, \quad \{ H_j \}_{ 0 \leq  j \leq m}$が
\textbf{同地}であるとは，$ n= m$で$ \{ 0 , 1 , n-1 \}$の適当な置換$ \sigma$によって
\[
	G_i / G_{i+1} \cong H_{\sigma(i)} / H_{\sigma (i) + 1} \quad ( 0 \leqq i \leqq n-1 )
\]
となることをいう．

\定理{2}
（Schreire の細分定理）

$ (G, \Omega )$の任意の正規列
$ \{ G_i \}_{0 \leq i \leq n} , \{ H_j \}_{0 \leq j \leq m}$に対し，
$ \{ G_i | \}$の細分 $\{ G_i \}$の細分$ \{ G_k' \} , \{ H_j\}$の細分$ \{ H_l' \} $で$ \{ G_k' \}$と$ \{ H_l ' \}$が同地名モノが存在する．



定理の証明には次の補題を用いる．


\補題{}
Zassenhaus）

$H,K$を$ ( G , \Omega )$の認容部分群，$ H' , K'$をそれぞれ$H , K$の認容正規部分群とする．そのとき，
$ H' ( H \cap K') , K' ( K \cap H')$はそれぞれ$ H' ( H \cap K)$および，$K' ( H \cap K)$の認容部分群で
\[
	H' ( H \cap K ) / H' ( H \cap K' ) \cong K' ( H \cap K ) / K' ( K \cap H')
\]

\証明

$H ' ( H \cap K )$は$H$の認容部分群である．第2同型定理によって$H' ( H \cap K) / H' \cong ( H \cap K ) / ( H' \cap K)$．
部分群$ H' ( H \cap K') / H' )$はこの同型対応によって$ ( H' \cap K) (H \cap K') / ( H' \cap K)$に写る．
$H \cap K' , H' \cap K$は$ H \cap K$の認容正規部分群だから，それらの積も同様である．
ゆえに上の同型により，$H ' ( H \cap K') / H'$は
$ H' ( H \cap K ) / H'$の認容正規部分群、従って$ (H' ( H \cap K')$は$ H' ( H \cap K)$の
認容正規部分群である．$ (H' ( H \cap K) / H') / ( H' ( H \cap K ' ) / H ' ) \cong ( ( H \cap K ) / ( H ' \cap K )) / 
(( H' \cap K)( H \cap K') / ( H ' \cap K))$であるから，第3同型定理によって
\[
	H' ( H \cap K ) / ( H' ( H \cap K' ) \cong ( H \cap K ) / ( H' \cap K ) ( H \cap K').
\]
右辺は$H , K$について対称だから，補題の同型が得られる．


[ 定理2の証明 ]

$ G_{ij}' = G_{i+1} ( G_i \cap H_j) \: ( 0 \leqq i \leqq n-1 , 0 \leqq j \leqq m), \: H_{ij}' = H_{j+1} ( H_j \cap H_i)
\: ( 0 \leqq j \leqq m-1 , 0 \leqq i \leqq n)$とおく．$ G_{ij'} \: ( 0 \leqq i \leqq n-1 , 0 \leqq j \leqq m-1)$および，
$ G_{n-1m}' - \{ e \}$は長さの$nm$の$ \{ G_i \}$の細分であり，$H_{ji}' \: ( 0 \leqq j \leqq m-1 , 0 < i < n-1)$および
$H_{m-1 n}' = \{e \}$は長さ$nm$の$ \{ H_j \}$の細分である．それが同知名湖とは補題ただちにわかる．


\定理{3}
（Jordan - H\"{o}lder）
$ G, \Omega )$が組成列をもてば，そのすべての組成列は同値である．


\問{1}
定理2を用いて，定理3を証明せよ．


定理3により，群が組成列 $ \{ G_i \}_{0 \leq i \leq n}$をもてば，$n$は一定である．これを
$(G , \Omega )$の\textbf{長さ}といい，$l_\Omega (G)$あるいは略して$l(G)$で表わす．


\例{}

$l(G) = 1$は$G$が単純ということと同値である．

\問{2}

簡約された正規列$ \{ G_i \}_{ 0 \leq i \leq n }$が組成列であるためには，剰余群$ G_i / G_{i+1} \: ( 0 \leqq i \leqq n-1)$が
すべて単純なことが必要十分である．

\証明

略

\問{3}

$ G, \Omega )$が組成列をもつならば任意の認容正規部分群 $N$を含むような
組成列が存在する．かつ $ l(G) = L ( N) + l ( G/ N )$．

\証明

略


\問{4}

同じ作用域を持つ群$G_1 , \cdots , G_n$がすべて組成列をもてば，それらの直積$ G= G_1 \times \cdots \times G_n$も
組成列をもつ．かつ$ l(G) = l(G_1) + \cdots + l(G_n)$．


\注意{}

任意の群が組成列をもつとは限らない．たとえば無限巡回群は組成列をもたない．

$ (G , \Omega )$の認容部分群全体の集合が包含関係について，極大（極小）条件をみたすとき，
$( G , \Omega )$は\textbf{極大}（\textbf{極小}）\textbf{条件}をみたすという．
$R$を単位環とし，$ (G, \Omega )$が$R$加群であるとき，極大条件をみたせば
\textbf{Neother加群}，極小条件をみたせば\textbf{Artin加群}という．


\問{5}

$ (G ,\Omega)$が極大条件をみたすことと，任意の認容部分群が有限個の元で生成されえることと
同値である．

\問{6}
$ ( G , \Omega )$が極大（極小）条件をみたせば，任意の認容部分群$H$も極大（極小）条件をみたす．
また，任意の認容正規部分群$N$に対して，$G / N$も極大（極小）条件をみたす．

\証明

略


\問{7}
同じ作用域$ \Omega $をもつ群$G_1 , \cdots , G_n$がみな極大（極小）条件をみたすならば，
$G = G_1 \times \cdots , \times G_n$もまた極大（極小）条件をみたす．

\証明

略


\問{8}

$(G,\Omega)$が極大条件および極小条件をみたせば，組成列をもつ．
逆に$ (G , \Omega )$が組成列をもてば，認容正規部分群について極大，極小条件が成り立つ．
