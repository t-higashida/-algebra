\subsection{剰余環，環の同型定理}


$ \mathfrak{a}$を環$R$の両側 ideal とすれば，$ x \sim y \Leftrightarrow x - y \in \mathfrak{a}$で定義された
関係は同値関係となる．
これで類別して得られる商集合$R/ \mathfrak{a}$において，元$x$を含む類を
$x\mod \mathfrak{a}$で表わす．

また，$( x \mod \mathfrak{a} ) = ( y \mod \mathfrak{a} ) $を$ x \equiv y ( \mod \mathfrak{a})$で表わす.

$\mathfrak{a}$ が両側 ideal であることから，
\begin{align*}
	&( x \mod \mathfrak{a} ) + ( y \mod \mathfrak{a} )  = (  x + y)  \mod \mathfrak{a}  \\
  & ( x \mod \mathfrak{a} )( y \mod \mathfrak{a} ) = ( xy  \mod \mathfrak{a} )
\end{align*}
によって，$R / \mathfrak{a}$に和および積が定義され，$ R / \mathfrak{a}$はかならずしも単位元をもたない
環となる．これを$R$の$\mathfrak{a}$による\textbf{剰余環}という．
$ \mathfrak{a} \ne R$ならば，$R / \mathfrak{a}$は単位元$1 \mod \mathfrak{a}$をもつ．

$f(1)=1'$をみたす準同型を，\textbf{単位元を単位元に写す準同型}という．
$R$から$R'$への準同型全射があるとき，$R'$は$R$に\textbf{準同型}であるという.
準同型単射を中への\textbf{同型写像}という．
$f:R \to R'$が中への同型写像ならば，$ x \in R$と$f(x) \in R'$とを同一視して，
$R$が$R$の中へうめこまれるという．とくに$R$から，$R$自身の中への準同型写像，
同型全射をそれぞれRの\textbf{自己準同型}，\textbf{自己同型}という．

\問{3}準同型写像
$f:R \to R'$が中への同型写像であるためには，$\Ker f= \{0\}$
の成り立つことが必要かつ十分である．

\証明
$ \Ker f = \{ 0 \}$とする．$f(x) = f(y) \Rightarrow f(x-y) = 0 \Rightarrow x -y \in \Ker f \Rightarrow x-y = 0 \Rightarrow x = y$
したがって，$f$は単射．

$f$が単射と仮定する．$0 \in \Ker f$は明らか．$ x \in \Ker f$とすると，$f(x) = 0' = f(0)$．よって$x=0$，つまり，
$x \in \{ 0 \}$，異状より，$\Ker f = \{ 0 \}$．
