\chapter{基本的な代数系}
\section{群}
\subsection{群の演算}

空でない集合$G$において，その任意の2元$x,y$に対して積$xy$を$G$の元として定める算法が与えられており，
次のG1〜3が成り立つとき$G$を\textbf{群}という．
\begin{list}{}{}
	\item	G1:　結合法則$(xy)z=x(yz)$が成り立つ．
	\item G2:　元$e$が存在して，$G$の任意の元$x$に対して$xe=ex=x$が成り立つ．
	\item G3:　$G$の任意の元$x$に対して，適当な元$y$をとれば，$xy=yx=e$が成り立つ．
\end{list}

\問{1}
群$G$の元$x_l,x_2,x_3,x_4, \cdots $に対して，$(x_1 x_2) x_3$を$x_1 x_2 x_3$，$ (x_lx_2x_3) x_4$を$x_1 x_2 x_3 x_4 \cdots $等と書く．
（一般に$ x_1 x_2 \cdots x_n = (x_1 x_2 \cdots x_{n-1}) x_n$によって，帰納的に$ x_1 x_2 \cdots x_n$を定義する．）
結合法則から次の\textbf{一般結合法則}が導かれることを，数学的帰納法によって示せ．

$m,n$を自然数とすれば，
\[
	(x_1 \cdots x_m) ( y_1 \cdots y_n) = x_1 \cdots x_m y_1 \cdots y_n.
\]

\証明
$n=1$なら定義そのもの，$n \geqq 2$の場合は結合法則と$n$に関する帰納法を用いて
\begin{align*}
	(x_1 \cdots x_m)(y_1 \cdots y_n) 
&	= (x_1 \cdots x_m)(y_1 \cdots y_{n-1} )y_n
	=	((x_1 \cdots x_m)(y_1 \cdots y_{n-1})) y_n \\
&	=	(x_1 \cdots x_m y_1 \cdots y_{n-1}) y_n = x_1 \cdots x_m y_1 \cdots y_{n-1} y_n.
\end{align*}

\問{2}
群において，$G2$の元$e$は一意的に定まる．

\証明
$e'$もまた$G2$の性質をもつとすると，$e' = e' e  = e$．

\問{3}
群$G$の元$x$が与えられたとき，$G3$の元$y$は一意的に定まる，

\証明
$xy = yx = e, xy' = y'x = e$ならば，$y=ey = (y'x)y= y' (xy) = y'e = y'$．

$G2$の$e$を$G$の\textbf{単位元}という．また，$G3$の$y$を$x$の\textbf{逆元}といい，
$x^{-1}$で表わす．

\問{4}
群の公理G1〜G3は，次のおのおのと同値である．

(1) $G1,G2l,G31$．(2) $G1,G2r,G3r$．(3) $G1,G2'$．
\begin{list}{}{}
	\item G2l:　元$e_l$（\textbf{左単位元}）が存在して，$ \forall x \in G , e_l x  = x$．
	\item G2r:　元$e_r$（\textbf{右単位元}）が存在して，$ \forall x \in G , x e_r  = x$．
	\item G3l:　$G$の任意の元に対して適当な元$y_l$（\textbf{左逆元}）をとれば，$y_lx = e_l$．
	\item G3r:　$G$の任意の元に対して適当な元$y_r$（\textbf{右逆元}）をとれば，$x y_r= e_r$．
	\item G2':　$ G$の任意の2元$x,y$に対して，$ux = y , xv = y$となる元$u,v$が存在する．
\end{list}

さらに，$e_l=e_r=e$，$y_l=y_r=x^{-1}$であること，および，$u,v$は$x,y$に対してただ1通りに定まることを証明せよ．

\証明
群の公理が成り立てば，あきらなに(1),(2)は成り立ち，また$u=yx^{-1}, v=x^{-1}y$とすることによって
(3)も成り立つ．逆に，(1)が成り立り立つとして，$x$の左逆元を$y_l$、$y_l$の左逆元を$z_l$とすれば，
\[
	x y_l = e_l (xy_l) = ( z_l y_l) = z_l ( y_l x_l) y_l = z_l e_l y_l  = z_l y_l = e_l.
\]
従って，$x e_l = x(y_l x) = (xy_l) x = e_l x = x$．ゆえに，$ e_l$は単位元で，$y_l$は$x$の逆元となる．

(2) が成り立てば，$G$は群になって，$e_r = e$，$y_r = x^{-1}$となることも同様に証明される．
(3) が成り立てば，$G$の一つの元$x_0$を任意に取るとき，$ex_0 = x_0$となる$e \in G$が存在する．

$G$の任意の元$x$は$=x_0 v$と表されるから，$ex = e ( x_0 v) = (e x_0) v = x_0 v = x$
ゆえに$e$は左単位元となる．また，$ux=e$となる．$ u \in G$が存在するから，任意の元$x$は
左逆元をもつ．ゆえに上に証明したことによって$G$は群となり，したがって$G2'$の$u,v$は一意的である．

\問{5}
$G \ni x$に対して，$n$個のxの積を$x^n$と書く．また，$x^0 = e, x^{-n} = ( x^{-1} )^n$と定めれば，
任意の整数$m$に対して$x^m$が定義される．このとき，\textbf{指数法則}
\[
	x^m x^n = x^{m+n}, \quad
	(x^m)^n =x^{mn} \quad (n,n \in \Z)
\]
が成り立つ．また，
\[
	xy = yx \quad \text{ならば} \quad (xy)^n = x^ny^n \quad ( n \in \Z)
\]

\証明
略．（書き下してみれば明らかであるが，厳密には，数学的帰納法による．）

群$G$において，さらに交換法則$xy=yx$も成り立つとき，$G$を\textbf{可換群}，また\textbf{Abel群}という．

可換群では，算法を加法の形$ x + y$で書くこともある．このときにはこの群を\textbf{加群}ともよぶ．
加群では，単位元を$0$，$x$の逆元を$-x$，$x^n$を$nx$で表わす．
従って指数法則は，
\[
				mx + nx = (m + n) x,
\quad		n(mx) = (nm) x,
\quad		n(x+y) = nx + ny
\]
と書かれる．

有限個の元から成る群を\textbf{有限群}といい，その元の個数を，この群の\textbf{位数}という．
無限に多くの元をもつ群は．\textbf{無限群}とよばれる．

\問{6}
$G$が有限集合ならば，群の公理 G1〜3 は次のおのおのとも同値である．

(4) $G1,G2l,G3'l$．(5) $G1,G2r,G3'r$．(6) $G1,G3'l,G3'r$．

ただし
\begin{list}{}{}
	\item	$G3'1$:　$G \ni x,y,z$ で $yx=zx $ ならば $y=z$．
	\item $G3'r$:　$G \ni x,y,z$ で $xy=xz $ ならば $y=z$，
\end{list}

\証明
群の公理 $ \Rightarrow $ (4),(5),(6)は明らかであるから，(4) $\Rightarrow$ (1), (5) $ \Rightarrow$ (2)，(6) $ \Rightarrow $ (3)を証明すればよい．
$ G3'l$が成り立てば，$G$の各元に$x$を右からかけるという写像は，$G$から$G$への単射である．
$G$は有限集合であるから，これはまた全射ともなる．したがって$G3l$が成り立つ．ゆえに(4) $ \Rightarrow $ (1)が証明された．
他の二つも同様に証明される．

\例{1}
ただ一つの元から成る集合$G = \{ e \}$において，$ee=e$によって結合を定義すれば，
$G$は群となる．この群を\textbf{単位群}という．

\例{2}
$ \Z,\Q,\R,\C$は，加法について可換群をなす．また，$\Q,\R,\C$からそれぞれ元0
を除いた集合は，乗法について可換群をなす．さらに，$\Q,\R$内の正数全体の集合も，
乗法について可換群をなす，二つの数1,-1から成る集合も乗法について可換群をなし,
絶対値1の複素数全体の集合も乗法について可換群をなす.

\例{3}
任意の集合$S$から$S$自身への全単射全体の集合を$\mathfrak{S} (S)$で表わす．
