\section{環と体}
\subsection{環の演算}

空でない集合$R$において，その任意の2元$x,y$に対して，和$x+y$および
積$xy$が$R$の元として定められており，次の R1 〜 3が成り立つとき，$R$を
環という．

\begin{list}{}{}
	\item	R1:　加法についてRは加群をなす（P.10参照）．
	\item R2:　乗法について結合法則$(xy)z=x(yz)$が成り立つ．
	\item R3:　左側分配法則$ x(y+z)==xy+xz $，右側分配法則 $(y + z ) x = yx+zx $が成り立つ．
\end{list}

\問{1} 左側分配法則から,左側一般分配法則
$ x (y_1 + y_2 \cdots + + y_n) = x y_1 + \cdots + x y_n$が導かれる．
右についても同様である．これを証明せよ．

\証明

$n$についての帰納法による．$x ( \sum_{i=1}^{n} y_i ) = x \sum_{i=1}^{n-1} y_i + x y_n = \sum_{i=1}^{n-1} x y_i + x_y = \sum_{i=1}^{n} x y_i$．
右についても同様．

\■

R1によって環$R$を加群と考えるとき，これを$R$の加法群という．その単位元を$R$の零元といい，$0$で表わす．

\問{2}
環$R$において，$ 0 x = x 0 = 0$，$ ( -x ) y = x ( -y) = - (xy) , (-x) (-y)  =xy$．

\証明

$ 0 x + 0 x = (0+0) x = 0 x$，両辺に$ -0 x$を加えて，$ 0x = 0$，$ x 0 = 0$も同様に証明される．
次に，$xy + (-x)y = ( x + (-x)) y = 0 y = 0$．ゆえに $ ( -x) y = - ( xy), x (-y) = - (xy)$も同様．
最後に，$ (-x) ( -y) = - (x ( -y)) = -(-(xy)) = xy$．

\■

環$R$の元$1$が，
\begin{list}{}{}
	\item	$1^\circ$ $1 \ne 0$
	\item $2^\circ$ $R$の任意の元$x$に対して$1 x = x 1 = x$
\end{list}
をみたすとき，$1$を$R$の単位元という．単位元をもつ環を単位環とよぶこともあるが，
本書では単位環のみを考えるから，これ単に環と呼ぶ．すなわち，
\begin{list}{}{}
	\item	R4: 単位元$1$をもつ．
\end{list}
ことを常に仮定する．$R$の元$x$に対して，$yx=1$をみたす元$y$が存在すれば，$y$を$x$の左逆元といい，
$xz=1$をみたす元$z$が存在すれば，$z$を$x$の右逆元という．また，
$xx^{-1} = x^{-1} x$をみたす元$z$が存在するとき，$x^{-1}$を$x$の\textbf{逆元}という．
左逆元をもつ元を\textbf{左可逆元}，右逆元をもつ元を\textbf{右可逆元}，逆元をもつ元$x$を\textbf{可逆元}または\textbf{単元}という．


\問{3} 単位元はただ1つである．$x$が左可逆でかつ右可逆ならば$x$は可逆である．
$x$の逆元$x^{-1}$は存在すればただ1つである．$x$が可逆元ならば
$x^{-1}$も可逆元で，$(x^{-1})^{-1} =x$．
$x,y$ともに可逆元ならば$xy$も可逆元で，$(xy)^{-1}=y^{-1}x^{-1}$.

\証明 群の場合と同様に証明される．

\■

問3により，単位元を持つ環$R$の可逆元全体の集合は積によって群をつくる．
これを$R$の乗法群といい，$R^*$で表す．環$R$において，
$y \ne 0, xy=0$が成立つとき，$x$を\textbf{左零因子}といい，
$y \ne 0 ,yx = 0$が成立つとき，$x$を\textbf{右零因子}という．
$0$は左右の零因子であるが，$R$が$0$以外に左零因子を持たないとき，
単に，$R$は\textbf{左零因子を持たない}という．また，$R$が
\textbf{左零因子をもつ}とは$0$以外の左零因子をもつことを意味する．
右零因子についても，同様の述べ方を採用する．

乗法についての交換法則$xy=yx$の成立つ環を\textbf{可換環}，
そうでない環を\textbf{非可換環}という．零因子を一つももたない
可換環を\textbf{整域}という．

\問{4}整域$R$において，$xy =xz$，$x \ne O$ならば$y=z0$．

\証明
$ xy = xz \Rightarrow x ( y-z) = 0$，$x \ne 0$であるから$ y -z$は零因子．$R$は整域であるから
$ y - z =0$，すなわち$y = z$．
\■

$0$以外の元がすべて可逆元である環を体という．乗法の交換法則が成り立つ体を可換体，
そうでない体を非可換体という．

\注意
可換体のみを体とよび，一般の体を斜体とよぶこともある．

\問{5} 可換体は整域である．

環$R$から環$R'$への全単射$f$があって，$R$の任意の元$x$，$y$に対して，
\[
	f ( x + y)  = f(x) + f(y), \quad f(xy) = f(x)f(y)
\]
が成り立つとき，$f$を同型写像という．同型写像が存在するとき$R$と$R'$は同型であるといって，
$R \simeq R'$で表わす．

\例{1}

$\C, \R, \Q$は，ふつうの加法と乗法とによって可換体となる．
$\Z$は体とはならないが，整域となる．

\例{2}
$R$を任意の環とするとき，$R$の元を係数とする文字$X$の多項式全体の集合を
$R[X]$で表わす．$R[X]$の$0$でない元$F(X)$は，
\begin{align}
	&F(X) = a_0 + a_1 X + a_2 X^2 + \cdots + a_n X^n, \\ \notag
  &a_0 , a_1 , \cdots , a_n \in R , a_n \ne 0
  \label{eq1x2x1x1}
\end{align}
の形に一意的に表わされる．ただし$n$は$0$または正の整数である．
$n$を$F(X)$の\textbf{次数}といい，$\text{deg} F(X)$で表わす．
$0$の次数は定めないが，次数$n$以下の多項式というときは，
$0$を含めるものとする．

$F(X)$はまた，$n$より大きい任意の自然数$m$を用いて，
\[
  F(X) = a_0 + a_1 X + \cdots + a_m X^m, \quad
  a_0,a_1, \cdots , a_m \in R
\]
の形にも書かれる．ここに$a_{n+1}= \cdots = a_m =0$である．

従って，$R[X]$の任意の二つの元$F(X),G(X)$が与えられたとき，
\begin{align*}
	F(X) &= a_0 + a_1 X + \cdots + a_m X^m, \\
	G(X) &= b_0 + b_1 X + \cdots + b_m X^m
\end{align*}
の形に書ける．このとき
\[
	F(X)+G(X) = (a_0 + b_0) + (a_1 + b_1)X + \cdots + (a_m + b_m) X^m
\]
と定義する．$m$は，$F(X),G(X)$のどちらの次数よりも小さくない整数であるが，
$m$をどう選んでも，結果として得られる$F(X) + G(X)$は同じである．あきらかに
\[
	\deg (F(X) +G(X)) \leqq \max (\deg F(X) , \deg G(X))
\]
が成りたつ．

次に，(\ref{eq1x2x1x1})の$F(X)$と
\[
	G(X) = b_0 + b_1 X + \cdots + b_q X^q, \quad b_q \ne 0
\]
との積は，
\[
	F(X)G(X)=a_0b_0 + \left( \sum_{i+j=1} a_ib_j\right)X
                  + \cdots 
                  + \left( \sum_{i+j=k} a_ib_j\right)X^k
                  + \cdots
                  + a_nb_q X^{n+q}
\]
によって定義する．これから
\[
	\deg (F(X)G(X)) \leqq \deg F(X) + \deg G(X).
\]
$0$と任意の多項式との積は$0$と定める．

このように定められた加法と乗法とによって，$R[X]$は環となる．これを$R$上の
\textbf{1変数多項式環}という．

とくに$R$が整域ならば，$ a_n \ne 0 , b_q \ne 0$から$ a_n b_q \ne 0$となり，$0$でない二つの多項式
$F(X),G(X)$について，
\[
	F(X)G(X) \ne 0, \deg (F(X)G(X)) = \deg F(X) + \deg G(X)
\]
が成り立つ．従ってまた$R[X]$も整域となる．

\例{3}
環$R$の元を係数とする文字$X_l,X_2, \cdots ,X_n$の多項式全体を$ R[ X_i, X_2, \cdots ,X_n]$
で表わす．$R [ X , \cdots , X_n]= R[ X_1, \cdots , ,X_{n-1} ][Xn]$であるから，$n$についての帰納法によって，
これもまた環となる．これを$n$変数多項式環という．$R$が整域ならば$R[Xl, \cdots ,Xn]$もまた整域となる．

\例{4}
$R$を環，$X_1, X_2 , \cdots , X_n$を文字とするとき，$ \sum_{0 \leqq i_\mu < \infty }^{} a_{i_1 \cdots i_n} X_1^{i_1} X_n^{i_n}$の
形の元を$R$上の$n$変数の形式的巾級数という．(すなわち，$\{ X_1, \cdots , X_n \}$から$\N \cup \{ 0 \}$への写像体の集合から
$R$への写像を$n$変数の形式的巾級数というのである．）
その全体は 例3と同様の方法で環となる．これを$R[[X_l, \cdots , X_n]]$で表わす．

\例{5}
環$R$の2元$x,y$に対して $x*y=yx$と定めれば，$+$を加法，$*$を乗法として
も$R$は環となる．この環をもとの環の対環とよび，$R^O$で表わす．$R^O$と同型な環は$R$
と\textbf{逆同型}であるという．

\例題{}

$K$を可換体とすれば，$K[X] \ni F(X) G(X) , G(X) \ne 0$に対して，
\[
	F(X) = G(X) Q(X) + R(X), 
\quad R(X) = 0 \quad \text{または} \deg R(X) < \deg( G(X)
\]
となるような$Q(X),R(X) \in K[X]$がただ1通り存在する．

\証明

$Q(X),R(X)$の存在を，$\deg F(X)$に関する帰納法で証明する．

$ \deg F(X) < \deg G(X)$のときは，$Q(X)=0,R(X)=F(X)$とすればよい．一般に，
$F(X) = \sum_{i=0}^{m} a_i X^i , G(X) = \sum_{j=0}^{n} b_j X^j$,
$ a_m \ne 0 , b_\ne 0. m \geqq n$とすれば，$F(X) - \frac{a_m}{b_m} X^{m-n} G(X)$の
次数は$m-1$以下であるから，帰納法の判定によって，
\[
	F(X) - \frac{a_m}{b_m} X^{m-n} G(X) = G(X) Q_1 (X) + R(X),
	\quad R(X)  0 \: \text{または} \: \deg R(X) < \deg G(X).
\]
$ Q(X) = Q_1(X) + \frac{a_m}{b_n} X^{m-n}$ とおけば，$Q(X), R(X)$は条件に適する．

次に，$Q(X),R(X)$の一意性を証明する．
\[
	F(x)=G(X) Q(X) + R(X) = G(X) Q'(X) + R'(X)
\]
がともに所要の条件をみたすとする．もし$R(X) \ne  R'(X)$とすれば，
$
	G(X) ( Q(X) - Q'(X) )
=R'(X)-R(X)$より，$Q(X)-Q'(X) \ne O$．従って$ \deg ( R'(X)-R(X)) = \deg G(X) + \deg (Q(X)-Q'(X)) \geqq \deg G(X)$となるが，
これは$R(X),R'(X)$についての条件と矛盾する．ゆえに$R(X)=R'(X)$．$K[X]$は整域で$G(X)=O$であるから，$Q(X)=Q'(X)$．

\問{6}
$R$を整域とすれば，$R[X] \ni F(X),G(X),G(X) \ne O$に対して，
\[
	b^s F(X) = G(X) Q(X) + R(X) \quad
	R(X)=0 \: \text{または} \: \deg R(X) < \deg G(X)
\]
となる$s \in \N \cup \{ 0 \} 2Vu{0},Q(X),R(X) \in R[X]$が存在する．
ただし$b$は$G(X)$の最高次の係数である．そのような$S$を一つ定めれば，
$Q(X),R(X)$は一意的に定まる．また，$s$は，$s \leqq \max(0, \deg F(X) - \deg G(X) + 1)$に選ぶことができる．

以上のことを例題にならって証明し，次に，この結果から例題を導け．

\証明

例題1にならう．$F(X) = aX^m + \cdots , G(X) b X^n + \cdots , a \ne 0 , b \ne 0$．$ m \geqq n$ならば，
$ b \cdot F(X) - a X^{m-n} G(X)$の次数が$m-1$以下となるから，帰納法の仮定によって
$
  b^{S'} (B F(X) - a X^{m-n} G(X))
= G(X) Q_1 (X) + R(X), s= s' + 1 , Q(X) = Q_1(X) + ab^{s'} X^{m-n}
$
とおけばよい．$ s' \leqq \max (0, (m-1 ) - n+1)$ならば，$ s = s' + 1 \leqq \max ( 0 , m - n +1 )$である．
一意性の証明は例題1と同様．

\例{6}
可換環$R$の元を係数にもち，三つの変数$i , j, k$をもつ1次以下の多項式全体の集合
\[
	Q = \{ a + bi + c_j + dk ; a,b,c,d \in R \: \} 
\]
の2元の和は多項式としての和で定め，積は，多項式としての積において，
\[
	i^2= j^2 = k^2 =-1, \quad
	ij = -ji = k, 			\quad
	jk = -kj = i,				\quad
	ki = -ik = j
\]
のおきかえをして得られる$Q$の元で定めることにすれば，これによって$Q$は，
非可換環となる．$Q$を環$R$上の4元数環という．

とくに$R$が実数体$R$の部分体ならば，$x = a + b i + c j + dk \ne 0 \Rightarrow N = a^2 + b^2 + c^2 +d^2 \ne 0$だから，
この$x$に対して $ y = \frac{a}{N} - \frac{b}{N}i - \frac{c}{N}j - \frac{d}{N} k$とおけば，
$ xy = yx =1$となる．したがって，この場合，$Q$は非可換体となり，4元数体とよばれる．
